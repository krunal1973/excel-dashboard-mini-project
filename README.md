# Excel Dashboard Project

## Overview

This project is an interactive Excel dashboard analyzing employee salary and distribution by department.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Bar & Pie)
* Slicer (for filtering)

## Features

* Total Salary KPI
* Salary by Department (Bar Chart)
* Employee Distribution (Pie Chart)
* Interactive Department Filter (Slicer)

## Insights

* IT department has the highest total salary
* Employee distribution is balanced across departments
